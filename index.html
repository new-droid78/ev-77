PHASE 4: GAME PLAN STEPS

1

🧪 Power Core Setup

Wire up main battery → motor → ESC → throttle. Make car move, get base EV working.

2

🔌 Generator Wheel System

Attach generator to rear wheel (belt/friction). Measure output, add boost converter, route to reserve battery.

3

🔁 Smart Battery Switch

Arduino + voltage sensors monitor batteries. Auto-switch when main < 20V. Keep batteries isolated.

4

📺 Display & Feedback

OLED showing: Battery %, Active battery, Generator output, Charge status.

5

🧠 Test, Break, Fix, Test

Full charge run, watch switching, test motion-only charging (manual push/jack wheel spin).
